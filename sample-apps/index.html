Sample Apps

 

Stormpath sample applications show you how to implement our SDKs and get started with Stormpath. Every example is open source and available on GitHub – Pull requests welcome!

Java
Spring Framework Sample
See the Stormpath Java SDK in action in Tooter, a Twitter clone.
  • Demonstrates the Spring Framework and the Stormpath Java SDK
  • Requires Maven

Apache Shiro Sample
For Apache Shiro users, Stormpath offers a full integration and sample web app.
  • Full application security and user management
  • Extends Shiro with user account, roles and permissions support
  • Requires Maven

Spring Security Sample
Login users to Spring Security apps with this plugin.
  • Full application security and user management
  • Uses standard or custom authentication filters and role checks
  • Requires Maven

Passport.js + Express.js Sample
A simple site built using Express.js, Passport.js, and Stormpath.
  • Demonstrates the Stormpath Passport.js integration.
  • Requires Express.js.
Download ZIP View on GitHub
Express.js API Auth Sample
A sample app using Stormpath and Express.js to secure a REST API.
  • Assign API keys to users.
  • Generate, manage, and revoke API keys.
  • Authenticate users with HTTP Basic Auth or OAuth2.

Stormpath + Heroku Sample
Instantly deploy a fully functional web app (with user registration) to Heroku!
  • Full application security and user management.
  • Can be instantly deployed to Heroku in one click.
  • Requires Express.js.

Chirper – Django Sample
