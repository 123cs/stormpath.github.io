Stormpath Java Servlet Plugin¶

The Stormpath Java Servlet Plugin is a drop-in plugin for web applications deployed to a Servlet container, like Tomcat or Jetty. It makes it incredibly simple to add user management, authentication and authorization to your application.

And the best part? You don’t even need a database!

This part of the documentation will show you how to get started with the Stormpath Java Servlet Plugin. If you’re new to the plugin, start here!
