Spring Boot Stormpath Web Starter¶

The Spring Boot Stormpath Web Starter allows you to use Stormpath almost instantly in any Spring Boot application. It makes it incredibly simple to add user management, authentication and authorization to your application.

Spring Boot Stormpath Web Starter helps automate all user registration, login, authentication and authorization workloads. It is very flexible - use only what you need or leverage the entire feature set.

And the best part? You don’t even need a database!

This part of the documentation will show you how to get started right away. If you are building a Spring Boot application and are new to Stormpath, start here!
