AuthenticationResult

An AuthenticationResult is returned when an application authenticates an account successfully.

Since: 0.1.1

method getAccount([options,] callback)

Retrieves the authentication result's corresponding successfully authenticated Account and provides it to the specified callback.

Performance Tip: The application authenticateAccount call will automatically request that the authenticated account be returned with the authentication result. This means the result.getAccount call will not actually send a request to the server: the account will be returned immediately.

Usage

result.getAccount(function(err, account) {
    if (err) throw err;
    console.log(account);
});

Parameters

ParameterTypePresenceDescription
optionsobjectoptionalName/value pairs to use as query parameters, for example, for resource expansion.
callbackfunctionrequiredThe callback to execute upon resource retrieval. The 1st parameter is an Error object. The 2nd parameter is the retrieved Account resource.

Returns

void; the retrieved Account resource will be provided to the callback as the callback's second parameter.
