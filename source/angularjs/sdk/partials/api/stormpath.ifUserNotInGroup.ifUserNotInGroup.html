ifUserNotInGroup
directive in module stormpath

Description

Use this directive to conditionally show an element if the user is logged in and is a member of the group that is specified by the string

Usage

as attribute
<ANY if-user-not-in-group>
   ...
</ANY>

Example

<div class="container">
  <h3 if-user-not-in-group="admins">Hello, {{user.fullName}}, please request administrator access</h3>
</div>
