$auth
service in module stormpath.authService

Description

The auth service provides methods for authenticating a user, aka "logging in" the user.

Methods

  • authenticate(credentialData)

    Sends the provided credential data to your backend server, the server handler should verify the credentials and return an access token which is store in an HTTP-only cookie.

    Parameters
    ParamTypeDetails
    credentialDataObject

    An object literal for passing usernmae & password, or social provider token

    Returns
    promise

    A promise which is resolved with the authntication response or error response (both are response objects from the $http service).

    Example

    Username & Password example
    myApp.controller('LoginCtrl', function ($scope, $auth, $state) {
  $scope.errorMessage = null;
  $scope.formData = {
    username: '',         // Expose to user as email/username field
    password: '',
  };

  // Use this method with ng-submit on your form
  $scope.login = function login(formData){
    $auth.authenticate(formData)
     .then(function(){
       console.log('login success');
       $state.go('home');
     })
     .catch(function(httpResponse){
       $scope.errorMessage = response.data.message;
     });
  }

});
