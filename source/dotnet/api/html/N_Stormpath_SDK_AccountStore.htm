Stormpath.SDK.AccountStore Namespace
The Stormpath.SDK.AccountStore namespace provides classes that are used to interact with AccountStore resources (Directories and Groups), and AccountStoreMapping resources.
Interfaces
  InterfaceDescription
IAccountStore
An abstract representation of a IDirectory or IGroup.
IAccountStoreMapping
Represents the assignment of an IAccountStore AccountStore (either a IGroup or IDirectory) to an IApplication.

When an IAccountStoreMapping is created, the accounts in the account store are granted access to become users of the linked IApplication. The order in which AccountStores are assigned to an application determines how login attempts work in Stormpath.
