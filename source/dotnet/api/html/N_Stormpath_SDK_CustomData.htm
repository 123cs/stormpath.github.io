Stormpath.SDK.CustomData Namespace
The Stormpath.SDK.CustomData namespace provides classes that are used to interact with Custom Data resources.
Interfaces
  InterfaceDescription
ICustomData
A dictionary resource embedded within another resource (such as an IAccount, IGroup, or IDirectory) that allows you to store arbitrary name/value pairs.
IEmbeddedCustomData
Provides access to convenience methods that manipulate the custom data of the parent resource.
