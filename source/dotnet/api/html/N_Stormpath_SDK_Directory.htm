IDirectory
A directory is a top-level container of IAccounts and IGroups. Accounts and groups are guaranteed to be unique within a directory, but not across multiple directories.

You can think of a IDirectory as an account 'store'. You can map one or more directories (or groups within a directory) to an IApplication. This forms the application's effective 'user base' of all IAccount that may use the application.

IDirectoryCreationOptions
Represents options for an IDirectory creation request.
