IClient Interface
The main entry point into the Stormpath C# SDK. To communicate with the Stormpath REST API from your project, you must first build a IClient instance. After obtaining an instance, the REST API may be used by making simple calls on objects returned from the IClient (or any child objects).

InstantiateT
Instantiates and returns a new instance of the specified IResource type. The instance is merely instantiated and is not saved/synchronized with the server in any way.

This method effectively replaces the new keyword that would have been used otherwise if the concrete implementation was known (Resource implementation classes are intentionally not exposed to SDK end-users).

(Inherited from IDataStore.)
VerifyAccountEmailAsync
Verifies an account's email address based on a sptoken parameter embedded in a URL found in an account's verification email.

For example:

Copy
https://my.company.com/email/verify?sptoken=ExAmPleEmAilVeRiFiCaTiOnTokEnHeRE

Based on this URL, the following should be invoked:

Copy
tenant.VerifyAccountEmailAsync("ExAmPleEmAilVeRiFiCaTiOnTokEnHeRE");

(Inherited from ITenantActions.)
