Stormpath.SDK.Tenant Namespace
The Stormpath.SDK.Tenant namespace provides classes that are used to interact with Tenant resources.
Interfaces
  InterfaceDescription
ITenant
A tenant represents a customer's private data 'space' within Stormpath that contains all of the customer's Stormpath-stored resources, like IApplications, IDirectorys, and IAccounts.
ITenantActions
Represents common tenant actions that can be executed on a ITenant instance or a IClient instance acting on behalf of its current tenant.
