Stormpath Angular JS SDK Ng Docs

Welcome!

This is the API documentation for the Stormpath AngularJS SDK. Here you will find the detailed description of all the compontents that this module provides to your application. With this SDK you can:

  • Authenticate users
  • Get the current user object
  • Know if the user is logged in
  • Use [ui-router][ui-router] to control access to your view states (authentication and authorization)
