Stormpath Python SDK¶

The official Stormpath SDK for Pythonistas!

An advanced, reliable and easy-to-use user management API, built by Python security experts.

Stormpath is a complete user management API. This library gives your Python app access to all of Stormpath’s features:

  • Robust authentication and authorization.
  • Schemaless user data and profiles.
  • A hosted login subdomain, for easy Single Sign-On across your apps
  • Social login with Facebook and Google OAuth.
  • Secure API key authentication for your service.

If you have feedback about this library, please get in touch and share your thoughts! support@stormpath.com

If you’d like to view our more comprehensive Python guide (which walks through every possible use case), you should probably check out the official Python SDK Product Guide.

For reference, the official Stormpath website can be found here: https://stormpath.com
