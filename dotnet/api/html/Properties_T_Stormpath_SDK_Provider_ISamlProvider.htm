Returns the PEM-formatted X.509 certificate used to validate the SAML Identity Provider's signed SAML assertions.
SsoLoginUrl
Returns the URL at the SAML Identity Provider where end-users should be redirected to login. This is often called an "SSO URL", "Login URL" or "Sign-in URL" for the Identity Provider (IdP).
SsoLogoutUrl
Returns the URL at the SAML Identity Provider where end-users should be redirected to logout of all applications. This is often called a "Logout URL", "Global Logout URL" or "Single Logout URL" (SLO).
