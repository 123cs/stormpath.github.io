SetDefaultAccountStore
Sets whether or not the associated IAccountStore is designated as the Application's default account store.

A value indicates that any accounts created directly by the application will be dispatched to and saved in the associated IAccountStore, since an application cannot store accounts directly.

If you use this setter, you will invalidate the cache for all of the associated Application's other AccountStoreMappings.

SetDefaultGroupStore
Sets whether or not the associated IAccountStore is designated as the Application's default group store.

A value indicates that any groups created directly by the application will be dispatched to and saved in the associated IAccountStore, since an application cannot store groups directly.

If you use this setter, you will invalidate the cache for all of the associated Application's other AccountStoreMappings.

SetListIndex
Updates the zero-based order in which the associated IAccountStore will be consulted by the linked IApplication during an account authentication attempt.

If you use this setter, you will invalidate the cache for all of the associated Application's other AccountStoreMappings.
