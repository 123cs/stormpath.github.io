SyncApplicationExtensionsAddAccountStore Method (IApplication, IAccountStore)
Synchronously adds a new IAccountStore to this Application and appends the resulting IAccountStoreMapping to the end of the Application's AccountStoreMapping list.

If you need to control the order of the added AccountStore, use the CreateAccountStoreMapping(IApplication, IAccountStoreMapping) method.
