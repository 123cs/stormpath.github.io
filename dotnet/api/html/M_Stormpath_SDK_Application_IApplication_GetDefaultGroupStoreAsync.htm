IApplicationGetDefaultGroupStoreAsync Method
Gets the IAccountStore used to persist new groups created by the application, or if no account store has been designated.

Stormpath's current REST API requires this to be a IDirectory. However, this could be a Group in the future, so do not assume it is always a Directory if you want your code to be function correctly if/when this support is added.
