Class: Directory

Directory

new Directory(directoryResource)

Encapsulates a Directory resource. For full documentation of this resource, please see REST API Reference: Directory.

This class should not be manually constructed. It should be obtained from one of these methods:

  • Account.getDirectory()
  • AccountStoreMapping.getAccountStore()
  • Client.getDirectory()
  • Client.getDirectories()
  • Group.getDirectory()
Parameters:
Name Type Description
directoryResource Object

The JSON representation of this resource, retrieved the Stormpath REST API.

getAccounts(collectionQueryOptions, callback)

Get the account's collection, which is a list of all the accounts in this Directory.

Parameters:
Name Type Description
collectionQueryOptions CollectionQueryOptions

Options for querying, paginating, and expanding the collection. This collection supports filter searches and the following attribute searches: createdAt, email, givenName, middleName, modifiedAt, surname, status, username.
callback function

The function to call when the operation is complete. Will be called with the parameters (err, CollectionResource). The collection will be a list of Account objects.
Source:
resource/Directory.js
Example
var query = {
  email: 'foo@example.com'
};

directory.getAccounts(query, function(err, collection) {
  collection.each(function(account, next) {
    console.log('Found account for ' + account.givenName + ' (' + account.email + ')');
    next();
  });
});
