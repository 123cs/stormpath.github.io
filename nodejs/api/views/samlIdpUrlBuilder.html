SamlIdpUrlBuilder

Provides the ability to build SAML IDP redirect URLs. This is done when Stormpath is initiating a redirect to a SAML IDP.

For more informaiton, please see Authenticating Against a SAML Directory.

constructor SamlIdpUrlBuilder(application)

Creates a new SamlIdpUrlBuilder instance for the provided application.

Usage

var builder = new stormpath.SamlIdpUrlBuilder(application);

Parameters

ParameterTypePresenceDescription
applicationApplicationrequiredStormpath Application that will issue the redirect. This application must be mapped to the relevant SAML Directories.

Returns

A new SamlIdpUrlBuilder instance.

method build([options,] callback)

Builds a SAML IDP URL and provides it to the specified callback.

Usage

builder.build(function(err, url) {
  if (err) {
    console.error(err);
    return;
  }
  console.log(url);
});

Parameters

ParameterTypePresenceDescription
optionsobjectoptionalOptional claims for the SAML Authentication JWT. Use these claims to control the callback url (cb_url), token state (state), and account store target (ash or onk). The required claims will be set automatically.
callbackfunctionrequiredThe callback to execute upon server response. The 1st parameter is an error. The 2nd parameter is an IDP redirect URL (string).

Returns

If the request fails, the callback's first parameter (err) will report the failure. If the request succeeds, a IDP redirect URL will be provided to the callback as the callback's second parameter.
