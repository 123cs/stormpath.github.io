StormpathAssertionAuthenticator

Provides the ability to authenticate with Stormpath JWTs (stormpath_token). Your application will recieve these tokens when a user is redirected to your application from a SAML provider, or from ID Site. This token asserts the user that authenticated.

constructor StormpathAssertionAuthenticator(application)

Creates a new StormpathAssertionAuthenticator instance for the provided application.

Usage

var authenticator = new stormpath.StormpathAssertionAuthenticator(application);

Parameters

ParameterTypePresenceDescription
applicationApplicationrequiredStormpath Application to authenticate against.

Returns

A new StormpathAssertionAuthenticator instance.

method authenticate(stormpath_token, callback)

Authenticates a stormpath_token and returns a AssertionAuthenticationResult, which can provide the Account that has authenticated.

The stormpath_token is the value of the jwtResponse parameter in the callback URL, e.g. https://myapp.com/samlCallback?jwtResponse=<stormpath_token>.

Usage

authenticator.authenticate(stormpath_token, function(err, authenticationResult) {
  if (err) {
    console.error(err);
    return;
  }
  console.log(authenticationResult);
});

Parameters

ParameterTypePresenceDescription
stormpath_tokenstringrequiredA Stormpath JWT, from a ID Site Callback or SAML Callback.
callbackfunctionrequiredThe callback to execute upon server response. The 1st parameter is an error. The 2nd parameter is an AssertionAuthenticationResult instance.

Returns

If the request fails, the callback's first parameter (err) will report the failure. If the request succeeds, a AssertionAuthenticationResult instance will be provided to the callback as the callback's second parameter.
